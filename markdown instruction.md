## Списки

Чтобы добавить ненумерованные списки, необходимо пункты выделить звездочкой (*). Например, аот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, пнкты необходимо просто пронумеровать.Например, вот так:
1. Первый пункт
2. Второй Пункт
